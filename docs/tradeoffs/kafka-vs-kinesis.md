# Kafka vs AWS Kinesis

## Contexto

A plataforma necessita suportar processamento de eventos em larga escala para casos de uso de marketing e Customer 360.

---

## Alternativas Avaliadas

### Apache Kafka

Vantagens:

- Open source
- Multi-cloud
- Grande ecossistema
- Independência de fornecedor

Desvantagens:

- Maior responsabilidade operacional

---

### AWS Kinesis

Vantagens:

- Operação simplificada
- Integração nativa AWS

Desvantagens:

- Vendor lock-in
- Menor portabilidade

---

## Decisão

Kafka foi selecionado por oferecer maior flexibilidade estratégica e menor dependência de fornecedor.

---

## Trade-off

Aceitamos maior complexidade operacional em troca de maior independência tecnológica.
