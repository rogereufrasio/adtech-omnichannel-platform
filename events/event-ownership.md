# Event Ownership

| Domain     | Team Owner                |
| ---------- | ------------------------- |
| Navigation | Digital Experience Team   |
| Commerce   | Commerce Team             |
| Customer   | CRM Team                  |
| Marketing  | Marketing Technology Team |

## Governance Rules

* Every event must have an owner.
* Every event must have a schema.
* Breaking changes require architecture review.
* Schemas must be versioned.
